诚聘英才

　网页设计美工

　　基本需求

  • 工作性质：全职或兼职
  • 工作地点：苏州
  • 招聘人数：2
  • 工作经验：不限
  • 学　　历：大专以上

　　岗位职责

  • 根据网站项目要求负责项目有关美术方面的设计与制作；
  • 负责美工技术及资源的整理；
  • 配合网站项目经理制定工作进度。

　　职位要求

  • 美术、平面设计、广告等相关专业，大学专科及以上学历；
  • 手绘能力强者优先，熟练掌握Photoshop，CorelDranw，llustrator、InDesign等图形设计软件以及能熟练运用网页三剑客Flash、Fireworks、Dreamweave等有关设计软件操作. 懂得HTML,DIV+CSS，具有扎实的美术基础、色彩控制能力强、想象力丰富；
  • 积极肯干，吃苦耐劳；具有创新精神及团队协作精神；
  • 责任心强，有团队合作精神，有良好的沟通能力;
  • 富有创意及执行力，有责任感、表达能力强。

　网络系统程序员

　　基本需求

  • 工作性质：全职
  • 工作地点：苏州
  • 招聘人数：3
  • 工作经验：不限
  • 学　　历：大专以上

　　岗位职责

  • 根据网站项目需求开发程序；
  • 整理书写程序相关文档资料；

　　职位要求

  • 熟悉具有Java EE项目开发经验；
  • 精通JavaEE各种标准组件与相关技术： JDBC, Servlet, JSP, EJB, XML, Web Services规范与应用开发；
  • 精通JavaEE开源框架：Hibernate/iBatis, Struts/Webwork, Spring；
  • 熟悉HTML/CSS/JavaScript和Ajax;
  • 熟悉mysql，sqlserver，oracle等数据库;
  • 具有团队合作意识，较强责任感和表达能力。

　系统推广销售人员

　　基本需求

  • 工作性质：全职
  • 工作地点：苏州
  • 招聘人数：5
  • 工作经验：不限
  • 学　　历：大专以上

　　岗位职责

  • 推广销售行业网络系统；

　　职位要求

  • 有一定网络基础，有良好的沟通能力，有敬业精神有良好的抗压能力与自律性；
  • 为人诚实，听从公司安排，有社交能力，喜欢与人沟通，有相关工作经验者优先。
